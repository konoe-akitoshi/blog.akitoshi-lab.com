# UI/UX Laws

ユーザビリティとユーザーエクスペリエンスに関する法則。

## 認知・操作に関する法則

### ヤコブの法則
慣習的UI（ハンバーガーメニュー等）を踏襲

### ヒックの法則
メニュー数を絞り選択肢を整理

### フィッツの法則
大きめのボタン、重要要素を近くに配置

### ミラーの法則
メニューを7±2項目にチャンク化

## システム設計の法則

### パレートの法則
よく使う機能に注力、他は簡略化

### テスラーの法則
ユーザーの負担を減らしシステムで複雑さを処理

## 心理・認知効果

### 美的ユーザビリティ効果
ビジュアル品質を上げて心理的満足感を向上

### 孤立効果
重要ボタンの色や形を周囲と差別化

### ピークエンドの法則
フィードバックや完了画面をポジティブに設計
